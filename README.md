# Claude Skills Guide (繁體中文版)

歡迎來到《Claude 技能指南》。這是一份從底層邏輯出發，旨在為開發者提供構建與管理 AI 驅動技能 (Skills) 的完整技術框架與方法論。

## 📖 關於本指南
本指南源自於 Anthropic 官方技術文檔的深度轉化。它不只是關於提示詞技巧的彙編，更是一份從架構、設計、測試到部署的完整技術指南。我們將探討如何將 LLM 的潛力轉化為具備自主性、可預測性與高效能的「技能 (Skills)」。

## 🚀 核心價值
透過本指南，您將學習如何將複雜的任務拆解為可執行的模組，建立穩定的 Agentic Workflow，並將 AI 技能整合進真實的生產環境。
